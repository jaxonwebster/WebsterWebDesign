# Freelance Web Design & Development
**Deployed Website:** [websterwebdesign.com](https://websterwebdesign.com/)

Welcome to the official repository for my freelance web design business! I build beautiful, responsive, and highly functional websites tailored to help individuals and small businesses establish a stunning online presence.

This repository contains the complete frontend architecture, custom styling, and page layouts for my professional portfolio and services website.


---

## 📂 Repository Structure


*   **`images/`** – Contains all visual assets, logos, and graphics used across the site.
*   **`styles/`** – Holds custom CSS stylesheets that define the typography, layouts, and overall visual theme.
*   **`index.html`** – The main landing page welcoming visitors and summarizing the brand.
*   **`services.html`** – A detailed breakdown of design packages, development options, and pricing.
*   **`mywork.html`** – A curated portfolio showcasing past client projects and web design case studies.
*   **`endorsements.html`** – Testimonials and reviews from satisfied clients.
*   **`quote.html`** – An interactive intake/contact form for potential clients to request a project estimate.
*   **`CNAME`** – Configuration file ensuring the repository seamlessly maps to a custom domain.

---

## 🛠️ Technologies Used

*   **HTML5** – Semantic markup for excellent accessibility and SEO structure.
*   **CSS3** – Custom responsive layouts, flexbox/grid configurations, and sleek modern typography.
*   **GitHub Pages** – Reliable, fast, and secure hosting directly from this repository.

## 🚀 Features

*   **Fully Responsive** – Optimized for seamless viewing on desktops, tablets, and smartphones.
*   **Client Intake Funnel** – A dedicated quoting page (`quote.html`) designed to convert visitors into active leads.
*   **Social Proof** – Structured endorsements page (`endorsements.html`) to display client trust and success stories.
